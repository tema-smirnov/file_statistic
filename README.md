# file_statistic
Данный файл создан для поиснения программы.
Цель созданного приложения - визуализация данных по удержаниям из таблицы excel.

В данном репозиотрии числятся файл самого код, готовое приложение, файлы необходимы для того, чтобы визуализация соответствовала фирменном стилю компании (файлы с шрифтами и логотипом).
Приложение адаптировано под монитор 1920х1080, в случае открытия файла на компьютере с другим разрешением экрана, всё может сбиться, в частности функция сохранения графика - реализована с помощью imagegrab.
Данные в excel вымышленные, для демонстрации функционала

Использование: необходимо запустить код/приложение, после чего выбрать файл, данные из которого будут использованы, в ходе работы можно сменить файл, также было принято решение для новых сторудников в месяцы, когда они ещё не работали проставлять в таблице 0, вместо использования метода fillna(0), чтобы не засорять общий график скачками вниз при увольнениях
